# EasyBusy - Frontend

Client side of the EasyBusy appointment management system.  
Built with React and Vite.

---

## 🚀 Tech Stack
- React
- Vite
- Axios
- React Router
- CSS

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/mayabargig/FrontEnd_EasyBusy.git
cd FrontEnd_EasyBusy
```

Install dependencies:

```bash
npm install
```

---

## 🔗 Environment Variables

Create a `.env` file in the root directory and add:

```env
VITE_EASY_BUSY_BACKEND__URL=http://localhost:5000/api/v1
```

If using production backend (Heroku):

```env
VITE_EASY_BUSY_BACKEND__URL=https://easybusy-api.herokuapp.com/api/v1
```

---

## ▶️ Run the App

```bash
npm run dev
```

The app will run on:

```bash
http://localhost:5173
```
